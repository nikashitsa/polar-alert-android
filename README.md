# Polar Alert

![cover](https://raw.githubusercontent.com/nikashitsa/polar-alert-android/master/cover.png)

## Usage

1. Click "Connect"
2. Set min/max BPM and sound volume
3. Click "Start"
4. When your BPM goes out limit BPM, you get sound alert

### Supported devices

- Polar H10
- Polar H9
- Polar OH1
- Polar Verity Sense
